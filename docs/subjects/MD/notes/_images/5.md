# 螺纹联接与螺旋传动

## 螺纹

- （重点）螺纹的主要参数：大径 $d$、小径 $d_{1}$、中径 $d_{2}$、线数 $n$、导程 $p$、螺纹升角 $\psi$
	- $\tan \psi = \frac{s}{\pi d_{2}} = \frac{np}{\pi d_{2}}$
- （重点）细牙螺纹的特点：细牙螺纹牙形小，螺距小，升角小，自锁性好，小径大，强度高，但牙易磨损，不易经常拆卸

## 螺纹联接类型及特点

- 受拉螺栓连接：普通螺栓、螺钉、双头螺栓、（紧定螺钉）
- 受剪螺栓连接

## 螺纹联接的强度计算

- 联接的失效形式
	- 受拉螺栓：塑性变形、疲劳断裂
	- 受剪螺栓：剪断、压溃
	- 联接失效：滑移、离缝
