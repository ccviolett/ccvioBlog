# 第零章 绪论

## 测试的基本组成及作用

| 组成         | 作用                                                              | 知识点                                                |
| ------------ | ----------------------------------------------------------------- | ----------------------------------------------------- |
| 激励装置     | 作用于无法显现或者不明显的信息， 产生搭载信息的新信号（包含信息） |                                                       |
| 传感器       | 获取信号，把特定的信息转换成可用信号（电信号）                    |                                                       |
| 信号调理     | 对信号进行变换、隔离、滤波、放大，以便传输和处理                  | 电桥、调制解调、滤波                                  |
| 信号处理     | 对信号进行运算的过程                                              | DSP、相关函数、功率谱、相干函数、倒频谱、数字信号处理 |
| 数据显示记录 | 记录信号                                                          |                                                       |

::: tip
**信号和信息是什么关系？**

信号是可用于传输和处理的信息载体
:::
